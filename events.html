Game Labs

Weekly Virtual Game Labs for 2020 Spring Semester -

  • Every Friday from 12-1:30PM
  • Location: VGDC at CSUF Discord Server
  • We go over recent events related to the Game Development Industry, showcase the hard work from the game project teams,
    and host weekly workshops covering all aspects in Game Development following showcases! Find schedules for those below!

Sound and Music -

  • Date and Time: Pushed to Friday April 17th, 2020 2-4PM
  • Location: VGDC at CSUF Discord
  • Learn simple Music Theory to make your own music and songs!
  • Learn the basics of Wwise and FL Studio
  • If you would like to follow along, bring a laptop with Wwise, FL Studio, FL Studio compatible chiptune software, and Wwise cube demo
  • Links for resources listed above will be available through Discord! Contact us for any questions!
