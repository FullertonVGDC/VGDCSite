Intro to Unity + Git Workshop -

  • Date and Time: Friday February 28th, 2020 4-6PM
  • Location: CSUF Campus CS 200
  • Want to take the first step in programming Video Games in Unity? Come to our Intro to Unity + Git Workshop!
  • We wll have a hands-on portion covering the basics of the Unity Engine and scripting with C#.
  • For those working with multiple programmers, we will go over how to use Git Clients when working with Unity!
  • We will cover good practices of version control and how to handle merge conflicts!
  • All skill levels are welcome! Please bring a laptop with Unity, Visual Studio, and GitHub Desktop installed if you wish to participate!
  • Notes will be presented afterwards for Paid Members!

VGDC 24 Hour Game Jam: Slam Jam! -

  • Please be prepared and bring your own equipment (laptops, chargers, etc) to participate!
